## 🔄 Run Loop 流程图详解

### 基础流程（无 Tool）

```
┌─────────────┐
│  用户输入   │
│  "Hello!"   │
└──────┬──────┘
       │
       ▼
┌─────────────────────────┐
│   Agent 理解意图        │
│   (解析 instructions)   │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│   是否需要调用 Tool？   │
│      判断               │
└──────┬──────────────────┘
       │
       │ 否
       ▼
┌─────────────────────────┐
│   直接生成回复          │
│   (基于 instructions)   │
└──────┬──────────────────┘
       │
       ▼
┌─────────────┐
│  输出给用户 │
│  "Hi! How   │
│   can I     │
│   help?"    │
└─────────────┘
```

### 完整流程（带 Tool 调用）

```
┌─────────────┐
│  用户输入   │
│ "北京天气   │
│  怎么样？"  │
└──────┬──────┘
       │
       ▼
┌─────────────────────────┐
│   Agent 理解意图        │
│   - 识别需要天气信息    │
│   - 匹配到 get_weather  │
│     tool                │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│   是否需要调用 Tool？   │──────────┐
│      判断               │          │
└──────┬──────────────────┘          │
       │ 是                          │
       ▼                             │
┌─────────────────────────┐          │
│   提取 Tool 参数        │          │
│   city = "北京"         │          │
└──────┬──────────────────┘          │
       │                             │
       ▼                             │
┌─────────────────────────┐          │
│   执行 Tool             │          │
│   get_weather("北京")   │          │
│   → 返回结果            │          │
└──────┬──────────────────┘          │
       │                             │
       ▼                             │
┌─────────────────────────┐          │
│   Tool 结果返回         │          │
│   "北京晴，25°C"        │◀─────────┘
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│   整合 Tool 结果        │
│   生成最终回复          │
└──────┬──────────────────┘
       │
       ▼
┌─────────────┐
│  输出给用户 │
│  "北京现在  │
│   晴朗，    │
│   气温 25   │
│   度。"     │
└─────────────┘
```

### 多轮对话循环

```
┌──────────────────────────────────────────────────┐
│                  Run Loop 循环                    │
│                                                   │
│  ┌─────────┐                                     │
│  │ 用户输入 │                                     │
│  └────┬────┘                                     │
│       │                                          │
│       ▼                                          │
│  ┌─────────────────┐                             │
│  │ Agent 处理      │                             │
│  │ - 理解意图      │                             │
│  │ - 决定是否调用  │                             │
│  │ - 执行/生成     │                             │
│  └────┬────────────┘                             │
│       │                                          │
│       ▼                                          │
│  ┌─────────────┐    继续对话？     ┌──────────┐  │
│  │  输出结果   │ ────────────────→ │  是：循环 │──┘
│  └─────────────┘                   └──────────┘
│       │                                 │
│       │ 否：结束                        │
│       ▼                                 │
│  ┌─────────────┐                        │
│  │  对话结束   │                        │
│  └─────────────┘                        │
│                                          │
└──────────────────────────────────────────┘
```

### Handoff 流程（多 Agent 协作）

```
┌─────────────┐
│  用户输入   │
│ "我要退款"  │
└──────┬──────┘
       │
       ▼
┌─────────────────────────┐
│   TriageAgent           │
│   (接待员/分类)         │
│   - 识别：退款问题      │
│   - 决定：转交          │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│   Handoff 转交          │
│   TriageAgent →         │
│   SupportAgent          │
└──────┬──────────────────┘
       │
       ▼
┌─────────────────────────┐
│   SupportAgent          │
│   (专家/处理退款)       │
│   - 调用 refund tool    │
│   - 生成退款方案        │
└──────┬──────────────────┘
       │
       ▼
┌─────────────┐
│  输出给用户 │
│  "您可以... │
│   申请退款" │
└─────────────┘
```
